## Instalação

```bash

git clone https://github.com/lyndontavares/desafio-fullstack

cd ./backend
mvn clean install -DskipTests=true
cd ..

cd ./frontend
npm i
npm run prod
cd ..

docker-compose build
docker-compose up

open browser http://localhost:8080/

```
